**Byte triggers**
=================

This package offers simple objcts to deliver triggers on serial port, parallel port or
LSL streams.

Install
-------

.. tab-set::

    .. tab-item:: PyPI

        .. code-block:: console

            $ pip install byte-triggers

    .. tab-item:: conda-forge

        .. code-block:: console

            $ conda install -c conda-forge byte-triggers

    .. tab-item:: Source

        .. code-block:: console

            $ pip install git+https://github.com/fcbg-hnp-meeg/byte-triggers
